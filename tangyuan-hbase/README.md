# Home
------

### 1. 项目介绍

tangyuan-hbase是tangyuan框架中的HBase服务组件，提供对HBase的访问和结果处理的支持。

### 2. 版本和引用

当前最新版本：1.2.1

	<dependency>
	    <groupId>org.xson</groupId>
	    <artifactId>tangyuan-hbase</artifactId>
	    <version>1.2.1</version>
	</dependency>
